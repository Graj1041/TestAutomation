Question 1:(Valid Plaindrome Check)
Solution : ValidPalindrome.py
Description :
for given input w,x,y,z from user, we can check for a word having an even length or odd length w.r.t these three values x,y,z.

Forming word of an even length
---------------------------------
1. All of three x,y,z will be even values. (Valid case for a Palindrome)
2. Either two of these will be odd and one even value.

Forming word of an odd Length
----------------------------------
1. All of three x,y,z will be odd values.
2. Either two of them will be even and one odd value. (Valid Case for a Palindrome)


Test data
----------

1.  w = 8
    x = 4
    y = 2
    z = 2

    Valid Palindrome : AA2##2AA

2.  w = 7
    x = 2
    y = 2
    z = 3

    Valid Palindrome : B5***5B

3.  w = 8
    x = 3
    y = 3
    z = 2

    Input is not valid to create a Palindrome

4.  w = 7
    x = 3
    y = 3
    z = 1

    Input is not valid to create a Palindrome
